# Villa Cart
An E-commerce website where you can buy crafts-related products.

Deployed link: https://villacart.netlify.app/

<h2>Pages</h2>
<ul>
  <li>Home Page</li>
  <li>Product Page</li>
  <li>Single Product Page</li>
  <li>Login & Signup</li>
  <li>Cart Page</li>
  <li>Payment Page</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Searching</li>
  <li>Sorting</li>
  <li>Filtering</li>
  <li>Pagination</li>
</ul>

<h2>Tech Stacks</h2>
<ul>
  <li>React</li>
  <li>Chakra ui</li>
  <li>JSON Server</li>
</ul>
